4. Listas

  • <ul> → Lista não ordenada. 
    Exemplo de uso:
<ul>
    <li>Item</li>
</ul>
  • <ol> → Lista ordenada. 
    Exemplo de uso:
<ol>
    <li>Primeiro</li>
</ol>
  • <dl>, <dt>, <dd> → Lista de definições. 
    Exemplo de uso:
<dl>
    <dt>HTML</dt>
    <dd>Linguagem de marcação</dd>
</dl>
